Crie o design de uma plataforma web moderna e inovadora para descoberta e aluguel de espaços comerciais, representando a empresa São Paulo Participações, que possui mais de 30 anos de experiência no setor imobiliário brasileiro.

O design não deve parecer uma cópia da OLX ou do Airbnb, mas pode usar essas plataformas apenas como referência de experiência do usuário e lógica de marketplace.

A interface final deve ser original, criativa e visualmente única, com um layout moderno e inovador.

O foco principal da plataforma deve ser:

sistema de filtros inteligentes

recomendação personalizada

assistente por chatbot com IA

descoberta rápida de espaços

O objetivo é criar uma plataforma de nova geração para encontrar espaços comerciais de forma rápida e inteligente.

IMPORTANTE:
O layout deve ser diferente dos marketplaces tradicionais, com estrutura inovadora, elementos interativos e navegação intuitiva.

Use criatividade e inovação no design — crie algo ousado, moderno e inteligente.

Conceito da Página Inicial

A página inicial deve funcionar como um centro de descoberta inteligente.

Assim que o usuário entrar no site, ele deve ser guiado imediatamente para encontrar o espaço ideal.

Elementos principais da homepage:

área central de busca inteligente

sistema de filtros dinâmicos

assistente virtual com chatbot

recomendações automáticas de espaços

A interface deve parecer interativa, moderna e fácil de usar.

Sistema de Filtros (Elemento Principal)

O sistema de filtros deve ser o coração da plataforma.

Evite usar o formato tradicional de filtros laterais.
Em vez disso, crie algo mais inovador, como por exemplo:

painéis de filtros flutuantes

cartões de filtros expansíveis

filtros em formato de bolhas interativas

filtros horizontais inteligentes

seletores visuais de categorias

Filtros disponíveis:

localização

faixa de preço

tipo de espaço comercial

tamanho do espaço

capacidade de pessoas

infraestrutura (wifi, estacionamento, mobiliado etc.)

Os filtros devem atualizar os resultados em tempo real, sem recarregar a página.

A experiência deve parecer rápida, fluida e inteligente.

Chatbot com Inteligência Artificial (Alta Prioridade)

Adicionar um chatbot com IA que ajude o usuário a encontrar espaços mais rápido do que usando filtros manuais.

O chatbot deve:

entender linguagem natural

sugerir espaços automaticamente

aplicar filtros sozinho

ajudar o usuário durante toda a busca

Exemplo de interação:

Usuário digita:

“Quero um escritório pequeno em São Luís até 2000 reais.”

O sistema responde mostrando os melhores espaços filtrados automaticamente.

O chatbot deve parecer um assistente imobiliário inteligente.

Página de Resultados

Criar uma interface moderna para mostrar os resultados da busca.

Elementos principais:

Cards de espaços contendo:

imagem grande

preço

localização

tamanho do espaço

avaliação

botão de salvar favorito

Adicionar opções como:

visualizar no mapa

comparar espaços

ordenar por relevância

O layout deve parecer limpo, moderno e focado em dados importantes.

Página do Espaço

Criar uma página detalhada do espaço contendo:

galeria grande de imagens

descrição completa

lista de características

mapa interativo

preço e disponibilidade

Botões principais:

Agendar visita

Entrar em contato

Estilo Visual

Criar um design moderno e profissional.

Características:

interface minimalista

sombras suaves

cartões bem organizados

tipografia forte e moderna

animações sutis

Sugestão de cores:

branco

cinza claro

azul ou verde institucional

O design deve transmitir tecnologia, confiança e inovação no mercado imobiliário.

Foco de Experiência do Usuário

Priorizar:

descoberta rápida de espaços

filtragem inteligente

busca assistida por IA

navegação simples e intuitiva

A plataforma deve parecer mais inteligente e moderna do que marketplaces imobiliários tradicionais.
